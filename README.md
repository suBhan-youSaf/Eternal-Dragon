# Eternal Dungeon

A fun terminal-based RPG made with clean C++ code.  
Perfect for learning Object-Oriented Programming and a great addition to my LinkedIn portfolio.

## Features

- Turn-based combat against enemies and bosses
- Inventory system (potions, weapons, armor)
- Shop system – earn gold by defeating enemies and buy items
- Save game
- Colored terminal interface with health bars

## How to Run

1. Open the project folder in Command Prompt / Terminal
2. Compile the game with this command:

```bash
g++ -std=c++20 *.cpp -o EternalDungeon.exe
